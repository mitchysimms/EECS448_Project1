public function handleClick(evt:MouseEvent):void

If non-cleared piece is clicked, piece is cleared or mine is set off. If unclicked piece is shift-clicked, flag is toggled and flagCounter is updated. If numbered piece is clicked and that number of flags surround it, clicks all unclicked pieces around that piece.

pre — a piece is clicked
